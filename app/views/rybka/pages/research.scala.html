Memory consolidation is the formation of long-term memory from short-term memories. Despite various studies, the problem of memory storage in the brain is not yet solved, and molecular mechanisms of memory loading and reading (retrieval) are not yet fully understood.

The Rybka Project will research proteins ( CREB, AP-2, zif268, etc. ) responsible for memory consolidation, reconsolidation, and learning, as well as their expressions within the zebrafish model.

The proteins will glow with help of green fluorescent protein (GFP) transfection. Using the Casper transparent strain of zebrafish, we will be able to observe these proteins in real time as the fish learns while solving cognitive tasks.
